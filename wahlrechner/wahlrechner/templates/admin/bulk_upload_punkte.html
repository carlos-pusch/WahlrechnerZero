Punkte-Bulkimport für Wahlgrafiken

Hier können Sie mehrere PNG- oder HTML-Dateien hochladen, die Punktegrafiken für eine Wahl darstellen. Die Dateinamen müssen dem Schema <slug>__<grafikkennung>.png oder .html entsprechen.

  • Slug: der eindeutige URL-Identifikator der Wahl (z. B. kommunalwahl_by_2026_augsburg_srw).
  • Grafikkennung: einer der folgenden Werte:
    • Gesamtpunkte_nach_Parlamentswahl__-1_1
    • Punkte_nach_These_und_Parlamentswahl__-1_1

Beispiel: kommunalwahl_by_2026_augsburg_srw__Gesamtpunkte_nach_Parlamentswahl__-1_1.png

Hinweis: HTML-Dateien werden bevorzugt dargestellt, falls vorhanden; andernfalls wird das PNG-Bild verwendet.

Bereits vorhandene Dateien werden überschrieben (gleicher Dateiname).
